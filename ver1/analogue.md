# Аналогичные сервисы BPMN Online

## Похожие сервисы визуализации BPMN 2.0

### Официальные демо bpmn.io
- **bpmn.io Demo** — https://demo.bpmn.io — Официальная демонстрация библиотеки bpmn-js от создателей библиотеки
- **bpmn-js Viewer** — https://bpmn.io/toolkit/bpmn-js/ — Просмотрщик и редактор BPMN диаграмм

### Camunda
- **Camunda Modeler** — https://camunda.com/download/modeler/ — Десктопное приложение для моделирования BPMN
- **Camunda Web Modeler** — https://camunda.com/products/cloud-modeling/ — Веб-редактор BPMN

### Red Hat
- **Red Hat Process Automation Manager** — https://access.redhat.com/products/red-hat-process-automation-manager — Инструмент для моделирования бизнес-процессов

### Другие онлайн-сервисы
- **BPMN.io Online** — https://bpmn.io/toolkit/bpmn-js/ — Просмотрщик от создателей библиотеки
- **ProcessMaker** — https://www.processmaker.com/bpm-glossary/bpmn-online-modeler/ — Онлайн-моделирование процессов
- **Bizagi** — https://www.bizagi.com/platform/modeler — Моделирование процессов в облаке

### Инструменты с поддержкой BPMN
- **Visual Paradigm** — https://www.visual-paradigm.com/ — Универсальный CASE-инструмент с поддержкой BPMN
- **Lucidchart** — https://www.lucidchart.com/pages/bpmn — Онлайн-диаграммы с поддержкой BPMN
- **Creately** — https://creately.com/ — Визуальное моделирование с BPMN

### Библиотеки для разработчиков
- **bpmn-js** — https://github.com/bpmn-io/bpmn-js — Основная JavaScript-библиотека для BPMN
- **bpmn-moddle** — https://github.com/bpmn-io/bpmn-moddle — Парсер BPMN 2.0 XML
- **diagram-js** — https://github.com/bpmn-io/diagram-js — Основная библиотека для диаграмм

## Сравнение

| Сервис | Тип | Онлайн | Бесплатно |
|--------|-----|--------|-----------|
| bpmn.io Demo | Просмотрщик | Да | Да |
| Camunda Modeler | Редактор | Частично | Частично |
| BPMN Online (этот проект) | Просмотрщик | Да | Да |

## Использованная библиотека

В данном проекте используется **bpmn-js** версии 18.13.1 — это наиболее популярная JavaScript-библиотека для визуализации и редактирования BPMN 2.0 диаграмм в браузере. Библиотека разработана компанией Camunda и распространяется под открытой лицензией.
